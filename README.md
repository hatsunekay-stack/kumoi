# 🎸 Katie式 雲井ペンタトニック対応表 (Kumoi Pentatonic Mapper)

## 🔥 概要

このアプリは、**雲井ペンタトニック（R 2 ♭3 5 6）**をベースに、
コード上での使い分けとギター指板ポジションを可視化する **Streamlitアプリ** です。

ブルースやJAZZの文脈に「和風ニュアンス」を加えるための
**Katie式アプローチ** を実装しています。

---

## 🎯 コンセプト

通常のスケール学習ではなく、

* スケールを「どこで使うか」
* どんな「響きになるか」
* 指板上で「どこを押さえるか」

を一体化して理解することを目的としています。

---

## 🧠 Katie式アプローチ

### m7コード上

| 使用スケール    | 響き        |
| --------- | --------- |
| ルート雲井ペンタ  | ドリアン風     |
| 4度上の雲井ペンタ | 哀愁のエオリアン風 |

---

### 7コード上

| 使用スケール    | 響き              |
| --------- | --------------- |
| ルート雲井ペンタ  | JAZZブルース風       |
| 5度上の雲井ペンタ | ミクソリディアン風       |
| ♭2上の雲井ペンタ | オルタード風（ドミナント向け） |

---

## 🎼 雲井ペンタトニックとは？

**構成音：**

```
R - 2 - ♭3 - 5 - 6
```

例：C雲井ペンタトニック

```
C - D - E♭ - G - A
```

👉 ナチュラル6（A）が入ることで
「和風＋浮遊感」の独特な響きになります。

---

## 🖥️ 主な機能

### ✅ コード対応表示

* Cm7 / C7 などを選択
* 各コード上で使える雲井ペンタを表示

---

### ✅ スケール構成音表示

* 選択スケールの音一覧

---

### ✅ ギター指板ポジション（2種類）

#### ● 図版モード

```
◎ = ルート
● = スケール音
－ = 非スケール音
```

👉 直感的に「形」で理解できる

---

#### ● 詳細モード

* 音名 + 度数（例：C(1), D(2)）

👉 理論理解用

---

### ✅ 全スケール一覧

全ルートの雲井ペンタを一覧表示


---

## 🎸 実戦での使い方

### 基本

* 8割：ブルース / マイナーペンタ
* 2割：雲井ペンタ

👉 混ぜることで「意表をつく」

---

### コツ

* フレーズの終わりに入れる
* ロングトーンで使う
* ルート（◎）に戻る

---

### 例（Cブルース）

```
C - E♭ - G - A → C
```

👉 A（6）で雲井感を出す

---

## 🌏 コンセプトの核心

> ブルース = 地面
> 雲井 = 空気感

👉 地面は崩さず、空気だけ変える
